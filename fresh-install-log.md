# ubuntu 23.10
## install
### basic nvim
+ nvim using snap

### zsh setup
+ install zsh
+ install zsh dependencies
  + batcat
  + fd-find
  + tree
  + fzf
  + ripgrep
+ install oh-my-zsh
+ install nerd font (FiraCode)
+ install powerlvl10k
+ symbolic link .zshrc to .dotfiles/zsh/zshrc files

### tmux setup
+ install tmux
+ install tpm
+ symbolic link .config/tmux to .dotfiles/tmux folder
+ use tpm to install plugins

### python, pip
+ install python3
+ install pip
+ optional python
  + install pipx
  + install poetry
  + install black
  + install isort

### devtools
+ s
